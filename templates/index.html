IP Geolocation API

A high-performance API that returns GeoIP and ASN information for IP addresses.

API Endpoints

Get current IP information

GET /self

Returns geolocation information for your current IP address.

Get specific IP information

GET /{ip_address}

Returns geolocation information for the specified IP address.
